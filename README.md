# Vue To-do (w/ Vuex & Router)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Starts the JSON Server API
```
npx json-server --watch api/db.json
```

### Compiles and minifies for production
```
npm run build
```

# About
Boilerplate project for Vue w/ Router & Vuex.
